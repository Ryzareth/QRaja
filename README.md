#QRaja

Overview
QRaja adalah web sederhana untuk membuat QR Code dengan cepat. Tinggal masukkan teks atau tautan, klik, dan QR Code langsung muncul. Cocok untuk kebutuhan sehari-hari tanpa ribet.

Main features
Instant QR generation. Tampilan sederhana dan mudah dipahami. Responsif, bisa dipakai di desktop maupun ponsel. Bisa menyimpan hasil sebagai gambar.

Tech
Dibangun dengan HTML, CSS, dan JavaScript murni. Menggunakan library qrcode.js untuk pembuatan QR.

How to run

1. Clone repository ke mesinmu
   git clone [https://github.com/Ryzareth/QRaja.git](https://github.com/Ryzareth/QRaja.git)

2. Buka folder proyek dan jalankan
   buka file index.html di browser

3. Jika mau jalankan server lokal (opsional)
   jalankan npx live-server di folder proyek
